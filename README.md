## Running Service Locally
- Run on port 8080 using the following commands at the repo root:
   - Using Docker:
      - ```docker build -t c3a1-img .```
      - ```docker run -dp 8080:8080 c3a1```
   - Using Go: 
      - ```go run main.go ```
